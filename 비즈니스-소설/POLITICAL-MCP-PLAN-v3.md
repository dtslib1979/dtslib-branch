## 만점까지 남은 것 2개

### 1. 실데이터 연결 — 국회 공공데이터 API

```
열린국회정보 API (data.assembly.go.kr)
  → 법안 투표 기록 (의안정보시스템)
  → 국회 회의록 (발언 기록)
  → 무료 / API 키 발급 즉시 가능

연결하면:
  Φ드라이버 7축이 "추정치"에서 "실데이터"로 전환
  analyze_politician() 결과가 실제 투표 기록 기반
```

### 2. 외부 유저 첫 케이스 — 역방향 유통

```
지금 GitHub 퍼블릭 됨 → 손 뗀다

자연 확산 경로:
  레딧 r/korea / r/PoliticalScience
  해커뉴스 Show HN
  트위터/X 정치 계정 RT
  
→ 보좌관 / 연구자 / 기자 DM

→ "외부 유저 첫 케이스" 완성
```
