dtslib 사용 설명서

Mobile First Programmer를 위한 가이드

1. 시작하기

dtslib은 GitHub Pages 기반의 개인 브랜딩 플랫폼입니다. Claude Code를 통해 음성 명령으로 웹사이트를 구축하고 관리할 수 있습니다.

1 GitHub 계정 생성

github.com/signup에서 계정을 만듭니다.

2 템플릿 복제

이 보일러플레이트를 Import하여 나만의 레포를 만듭니다.

3 GitHub Pages 활성화

Settings > Pages에서 main 브랜치 배포를 설정합니다.

2. 레포지토리 구조

dtslib은 6개의 레포지토리로 구성됩니다:

  • branch - 메인 랜딩페이지 + Staff Portal
  • posts - HTML 블로그 포스트
  • logs - 캡처 데이터 저장소
  • pwa - PWA 앱 (Vercel 배포)
  • apk-lab - Android APK 빌드
  • media - 이미지/오디오 CDN

3. Control Panel 사용법

Writing 도구

  • GitPages - GitHub Pages 블로그 관리
  • HTMLPost - HTML 포스트 작성 및 배포
  • WebPrompt - AI용 웹페이지 프롬프트 생성

Media 도구

  • ImageUp - 이미지 업로드 (GitHub API)
  • AudioUp - 오디오 업로드 (GitHub API)

4. Claude Code 연동

Termux에서 Claude Code를 설치하고, Staff Portal의 인스트럭션을 복사하여 붙여넣으면 자동으로 레포가 구축됩니다.

음성 명령으로 코드 수정, 배포, 콘텐츠 작성이 가능합니다.
